1. find : 해당요소찾기 - 첫번째 요소만 찾는다(없으면 undefined)


2. find : 인덱스를 이용한 해당요소 찾기 -


const nums = [11,21,30,40,50];

3. find : 짝수 또는 홀수 값찾기


4. findIndex : 첫번째 요소의 인덱스위치 찾기 (없으면 -1)


5. filter : 조건을 만족하는 모든 요소를 배열형태로 반환
모든 사과 찾기 / 짝수 찾기


6. reduece : 모든 요소를 이용하여 하나의 값을 반환
합구하기 / 최대 / 최소


7. Map : 배열순회 후 콜백함수를 적용하여 각요소를 변환하고 새로운 배열을 만환한다. const fruits = ["apple", "BANAna", Cherry];


8. some : 조건을 만족하는 요소가 하나라도 있으면 true를 반환


9. every :배열의 모든 요소가 조건을 만족하면 true를 반환
